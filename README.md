# Vir-Craft

## Dependencies
- Python
- Ollama

## Credits
- Credits to Neal Agarwal for the original game "Infinite Craft"

## How to play
- You can right click to see a list of all the items you have
- You can send 2 items and fuse them, to get a new word
- The words are AI generated, hence there is no fixed "recipe", and results may vary with different users for the same fuse
- The goal is to get the wierdest word
- Will be adding a leaderboard in the future

## IMPORTANT
- Make sure all files given are in the same folder
- In the same folder, also create a file called "save.txt", "name.txt", and "Wierdness.txt"
